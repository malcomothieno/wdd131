Explore Uganda – Site Plan

Explore Uganda – Website Plan

1. Site Name

Explore Uganda

I chose this name because the website is focused on sharing places, culture, and travel ideas from different parts of Uganda.

2. Site Purpose

The purpose of the site is to help visitors discover popular attractions, local food spots, and cultural activities. It will also let users filter attractions and save favorites using JavaScript and localStorage.

3. Scenarios

  • “What interesting places can I visit in Uganda?”
  • “Where can I find local experiences like food or cultural spots?”

4. Color Scheme

Primary Color (#2c5f2d): Used for headings.

Secondary Color (#97bc62): Used for backgrounds and accents.

5. Typography

Montserrat – Headings
Open Sans – Paragraph text

6. Wireframes

  • Footer

Desktop Layout
